## Costo de la sesión
**Muy alto (~$500).** Cada ciclo implementer+reviewer costó bastante más de
lo esperado para tareas "bite-sized" — el overhead de contexto que cada
subagente relee (CLAUDE.md completo, brief, diff, reporte) se acumula rápido
en un proyecto grande con un CLAUDE.md extenso. Se pausó y confirmó con el
usuario en 5 checkpoints distintos de costo antes de continuar cada vez.

**Learning para la próxima ejecución grande con subagent-driven-development
en este proyecto:** considerar reviews inline (sin subagente) para tareas de
bajo riesgo (migraciones simples, cambios de frontend puramente visuales), y
reservar el ciclo completo implementer+reviewer para código que toca dinero,
auth, o RLS.
